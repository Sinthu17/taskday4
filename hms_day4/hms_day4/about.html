About Our Project

The Hospital Management System (HMS) is a web-based application designed to streamline hospital operations: patient records, doctor schedules, appointments, billing, and reports.

✨ Key Features

  • 👨‍⚕️ Manage Patients & Doctors
  • 📅 Schedule & Track Appointments
  • 💳 Billing & Payment Records
  • 📊 Generate Reports
  • 🌐 User-Friendly Interface with Bootstrap

🎯 Project Objectives

The goal is to build a practical system using HTML, CSS (Bootstrap), JavaScript and PHP backend with MySQL database, developed as part of an internship program.
